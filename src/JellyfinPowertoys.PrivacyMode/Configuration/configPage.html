Privacy Mode

This plugin adds a privacy mode that hides content when activated.

Controls

Activation

Double-tap any home row key (a, s, d, f, j, k, l, or ;) to activate privacy mode. This will apply a blur to hide content on the page.

Deactivation

Shift + double-tap the same activation key to deactivate privacy mode. This will return the page to its normal state.

Hover Reveal

Double-tap a different home row key (while privacy mode is active) to enable hover reveal. This will allow content to be revealed on mouse hover, with a slight delay.

Similarly to privacy mode deactivation, shift + double-tap the same used to activate hover reveal will deactivate hover reveal.
